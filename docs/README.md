# Documentation

This folder contains all documentation related to building and installing `droidrun[google]` in Termux on Android emulator.

## Files

### Installation Guides

- **`droidrun_dep_install_seq.md`** - Step-by-step installation sequence guide
  - Optimal order to install dependencies
  - Prevents multiple builds
  - Complete command reference

- **`droidrun-google-dependencies.md`** - Complete dependency tree and requirements
  - All dependencies (direct and transitive)
  - Build requirements for each package
  - Version constraints
  - Build time estimates

### Troubleshooting

- **`termux-build-errors-and-solutions.md`** - Comprehensive error documentation
  - All errors encountered during builds
  - Root causes and solutions
  - Prevention strategies
  - Quick reference fixes

### System Requirements

- **`emulator-requirements.md`** - Emulator configuration guide
  - RAM and storage requirements
  - Recommended settings
  - Configuration instructions

### Reference

- **`wheelbuildcmd.txt`** - Quick reference for building wheels
- **`termux-build-commands.txt`** - Common Termux build commands
- **`quick-wsl-commands.txt`** - WSL-related commands (if applicable)

## Quick Start

1. Read `droidrun_dep_install_seq.md` for the installation sequence
2. Check `emulator-requirements.md` to configure your emulator
3. Follow the sequence in `droidrun_dep_install_seq.md`
4. Refer to `termux-build-errors-and-solutions.md` if you encounter errors
